#Tictacky
Simple tic tac toe game (HTML, CSS &amp; jQuery).

___

## 2 player Tic tac toe game

+ Made using HTML, CSS and Javascript/jQuery.
+ Blue player vs pink player instead of Xs and Os.
+ Built with flexbox - responsive design.
